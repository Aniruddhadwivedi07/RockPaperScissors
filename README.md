## ✨ The Experience

Step into the arena with a beautifully designed, dark-themed **Rock Paper Scissors** game. It's not just a single round—it's a **Best of 5 Match**! Can you outsmart the bot and secure the crown?

---

## 🚀 Key Features

| Feature | Description |
| :--- | :--- |
| 🏆 **Best of 5 Match** | The first player to reach 3 wins takes the match crown! |
| 📊 **Live Scoreboard** | Real-time tracking of both Player and Computer scores. |
| 🔥 **Winning Streak** | Monitors your consecutive wins (resets instantly when the computer scores). |
| 📜 **Match History** | A scrollable, color-coded log of every round played in the current match. |
| 🎨 **Premium UI** | Stunning glassmorphism panels, deep space gradients, and smooth glowing micro-animations. |

---

## 🛠️ Tech Stack

- **React.js** - Robust state management using hooks (`useState`).
- **Vite** - Lightning-fast development server and optimized build process.
- **Vanilla CSS** - Custom-crafted CSS featuring modern layout techniques, gradients, animations, and custom scrollbars.

## 🏃‍♂️ How to Run Locally

Want to try it out on your own machine? It's incredibly easy!

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Aniruddhadwivedi07/RockPaperScissors.git
   ```

2. **Navigate into the project:**
   ```bash
   cd RockPaperScissors
   ```

3. **Install the dependencies:**
   ```bash
   npm install
   ```

4. **Spin up the development server:**
   ```bash
   npm run dev
   ```

5. **Play the game!**
   Open your browser and navigate to `http://localhost:5173` to start playing. Good luck! ✌️
